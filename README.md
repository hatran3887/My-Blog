# My Blog

A simple blog web app built with Flask. You can view, add, edit, and delete blog posts. Posts are stored in a JSON file.

## Features

- View all blog posts
- Add a new post
- Edit an existing post
- Delete a post

## Setup

1. Install the dependencies:

   ```
   pip install -r requirements.txt
   ```

2. Run the app:

   ```
   python app.py
   ```

3. Open your browser at [http://localhost:5000](http://localhost:5000).
